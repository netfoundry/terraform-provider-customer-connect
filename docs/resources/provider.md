# customerconnect_provider (Resource)

Manages a NetFoundry Customer Connect Provider — the top-level entity that owns connectors, connector models, locations and customers within a network.

## Example Usage

```terraform
resource "customerconnect_provider" "example" {
  name            = "NF Corp Provider"
  network_id      = "00000000-0000-0000-0000-000000000001"
  organization_id = "00000000-0000-0000-0000-000000000002"
}
```
